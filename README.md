# Kurban Kesim ve Hisse Takip Sistemi

Kurban kesim organizasyonu için hayvan, hissedar ve kesim takip sistemi.

## Özellikler

- **Kişi Yönetimi**: Kişi kaydı, ağırlık kategorisi seçimi, peşinat/ödeme takibi
- **Hayvan Yönetimi**: Küpe no, kilo, fiyat, hisse adedi takibi
- **Hisse Ataması**: Kişileri hayvanlara hissedar olarak atama
- **Kesim Paneli**: Sıralı kesim takibi, vekalet kontrolü
- **Dashboard**: Genel durum özeti
